Site CMS

Add posts, projects, skills, achievements, and certifications from a form — this saves straight to your GitHub repo, so the live site updates without you touching any code.

🔌 Connect to your repository

Uses the GitHub API directly from your browser. Your token is stored only in this browser's local storage and is sent only to api.github.com — never to Claude or anyone else.

Create a fine-grained token at github.com/settings/personal-access-tokens/new — limit it to this one repository, and grant Contents: Read and write permission only. Nothing else is needed.
